# Walmart Sales Forecasting 📊

## 📌 Project Overview
This project predicts weekly sales using Machine Learning techniques on retail data.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering (Year, Month, Week)
- Random Forest Model

## 📈 Results
- R² Score: 0.89
- MAE: 4131
- RMSE: 7167

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 💡 Key Insights
- Sales follow seasonal patterns
- Store-level trends impact performance
